# @zypc/vue-ui

## zypcInput-输入框

| 属性            | 说明             | 类型          | 默认值 |
| --------------- | ---------------- | ------------- | ------ |
| width           | 输入框宽度       | String        | 200px  |
| value           | 绑定的值         | String/Number | 空     |
| backgroundColor | 输入框的背景色   | String        | white  |
| placeholder     | 输入框中默认内容 | String        | 空     |

## zypcButton-按钮

| 属性 | 说明                                 | 类型   | 默认值 |
| ---- | ------------------------------------ | ------ | ------ |
| size | 按钮尺寸可选`small`/`large`          | String | small  |
| type | 按钮类型颜色`info`/`success`/`error` | String | info   |
